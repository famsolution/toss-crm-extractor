# 토스인슈 CRM 추출기 (Chrome 확장)

토스인슈 CRM 데이터 추출 + 보험료비교 폼 자동입력. **로그인 + 관리자 승인** 권한 게이트 포함.

## 📥 설치 방법

1. [**최신 버전 다운로드**](../../releases/latest/download/toss-crm-extractor.zip) (releases/latest)
2. 받은 `toss-crm-extractor.zip` **압축 해제**
3. Chrome 주소창에 `chrome://extensions` 입력
4. 우측 상단 **개발자 모드** 켜기
5. **압축해제하여 로드** → 압축 푼 폴더 선택

## 🔐 사용 (권한 필요)

1. 확장 아이콘 클릭 → **회원가입** (이메일/비밀번호)
2. 가입하면 자동으로 **승인 대기** 상태가 됩니다
3. **관리자 승인** 후 사용 가능 (관리자가 승인 토글을 켜면 즉시 활성화)

> 승인 전에는 추출·자동입력 기능이 차단됩니다.
